Introduction

Overview

This document provides an explanation of how to use the Training Plan REST API.

Request Structure

The requests for this API must use x-www-form-urlencoded requests.

Routes

Use the navigation at the side of this page to view information about the routes exposed by this API.

Date format

All times in this database are measured from 01 Jan 1970 00:00:00 UTC.

Editing this document

This document contains a directory called "/api". Contained within this is a directory structure which mirrors the routes of this API. In each API route's directory there is an info.json file containing the set of methods that the route accepts, as well as individual GET.json, PUT.json etc. files which specify information about the respective methods.
